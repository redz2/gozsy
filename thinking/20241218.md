# 前端 vs 后端
1. 前端能干什么？
    * 配置管理
    * 任务管理
        * 周期性任务
        * 一次性任务（虚拟机申请、权限申请）
        * 定时任务
    * 数据的展示、修改
2. 后端能干什么？
    * 读取系统配置，启动服务
    * 执行任务
    * 提供接口，用来展示、修改数据
        * 对数据进行分析、汇总，提供有用的信息
3. 更加直白的说法（后端像什么？）
    * 初始化: 机器应该长啥样
    * 操作工: 干各种活
    * 记事本: 记录信息，查询信息

4. 一些技巧
    * 以卫语句取代嵌套条件表达式 -> 简化代码结构
